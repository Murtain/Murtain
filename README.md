### 概述

Uranus 是包含常用模块的轻量快速开发应用程序框架。

### 设计目标

- [ ] ~模块化（Module）~
- [ ] 基于领域驱动设计的分层架构（DDD-NLayer）
- [ ] ~依赖注入（Denpendecy Injection）~
- [x] 对象映射（Object Mapper）
- [ ] 日志（Logging）
- [ ] 工作单元（UnitOfWork）
- [x] 配置管理（Configuration Management）
- [x] 缓存管理（Cache Management）
- [ ] 分布式锁（Distributed Lock）
